# Processed Data

This directory stores local demo data used by the backend.

- `hefei_pois.sqlite`: POI catalog loaded by `PoiRepository` when present.
- `ugc_hefei.jsonl`: nested UGC review corpus used by lexical fallback and embedding generation.
- `ugc_hefei.faiss`: FAISS `IndexFlatIP` index generated by `python scripts/embed_ugc.py`.
- `ugc_hefei_embeddings.npy`: optional BGE-small-zh embedding matrix kept for local fallback and inspection.
- `ugc_hefei_meta.jsonl`: metadata aligned row-by-row with the embedding matrix.
- `agent_sessions.sqlite`: runtime SQLite store for agent sessions, created by `app.agent.store`.

Regenerate UGC embeddings from the project root:

```powershell
python scripts/embed_ugc.py
```

`UgcVectorRepo` searches FAISS first. If the FAISS index is missing or unavailable, it falls back to the `.npy` embedding matrix, then to lexical search.
